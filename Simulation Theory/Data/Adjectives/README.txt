Rules For Data-sets:

The file should be title in the following way:
    length_adj_category_.txt

    file length = 10
    category = food_opinion

    For example: 10_adj_food_opinion.txt

The file should be laid out in the following way:
    Line 1 will be an example sentence.
    Then the list of words from most negative to most positive.
    All words must be lowercase with no unnecessary punctuation
    All words must substitute into the example sentence perfectly.

Please see adj_example.txt for the example file.
